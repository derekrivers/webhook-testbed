# three.js overview

three.js is a JavaScript library for building interactive 3D graphics in the browser. It helps developers create scenes with cameras, lights, geometry, materials, textures, animation, audio, and user interaction on top of WebGL.

In practice, three.js can be used for product visualizations, games, simulations, architectural walkthroughs, data visualizations, AR/VR-style experiences, and other animated 3D interfaces that run on the web. Its documentation shows support for core rendering features, reusable helpers, object loaders, multiple camera types, lighting systems, and many built-in geometry and material options.
